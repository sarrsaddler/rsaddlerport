Bilingual E-card

Learn how I created an email that served IBM's Bluemine native and Enligh speakers.

snippet of the email design.

Context

Design ask: Create an E-card that works for both of English and Japanese audience.

Team members: I was partnered with an analyst for this project. The analyst create the content. My task was to figure out how the content should be laid within the email.

Design Procress

How did I accomplished this project?

What I decided to do was think about where these ecard would be sent. The primary audience would be Japanses so I decided to place the Japanese content first. Above the Japanese content I place one line in English to inform English speakers that there is the English version below.

Coding

I was able to code this email within a IBM development environment tool called Silverpop. I used table tag and other html tags to design this layout.

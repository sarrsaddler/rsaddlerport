Overview
What is DAVE?

The Tableau environment was meant to build graphs, not create a web app experience. At the time, I was unfamiliar with using Tableau. I didn’t know the technical limitation of the tool. When trying to access Tableau's help center resource, I couldn't find content that would have aided me in the process of this redesign. DAVE's minimum viable product was also set to be launched later that week. Below I have listed my main impediments meanwhile completing this project.
  • The Tableau dashboard was unresponsive. The tool did not allow for a resizable layout.
  • I had to rely on inserting an image to look like a button within the dashboard. Also, I could not add any alternative text to the picture. This was concerning to me because of accessiblity.
  • The data scientist had a lot of content to review. Like 30 pages worth.

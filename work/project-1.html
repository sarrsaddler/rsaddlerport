DAVE

UX Design
Client
IBM’s Market Development & Insights organization
My Contributions
Duration: One work week (September 2019)
During this project my primary job role was being the UX designer.
Project Overview
IBM’s Market Development and Insight organization operates like an internal consultant. The organization provides IBM senior leadership with competitor analysis and the latest market intelligence. Due to the demand of MD&I's insight, marketers within the organization needed a way to build data visuals fast to include in their reports. MD&I gathered a team of data scientists to leverage the capabilities of Tableau, an interactive data visualization software. The data scientists created DAVE a set of dashboards meant to mimic pages within a website.
Context to the problem
DAVE was successful in helping marketers within MD&I create data visualizations but it was not user friendly. Marketers reported to the data science team that the user interface was confusing. They did not know where to seek education content to better understand the environment. DAVE needed improvement fast because it was set to launch within one work week.
‍
‍
Design Prompt
What was I asked to do?
  • Improve the navigation of the DAVE tool to increase user engagement, better communicate the benefits, and drive repeat usage.
  • Add visual consistency to the user interface.
Challenges
What were my impediments during this project?
The Tableau environment was meant to build graphs, not create a web app experience. At the time, I was unfamiliar with using Tableau. I didn’t know the technical limitation of the tool. When trying to access Tableau's help center resource, I couldn't find content that would have aided me in the process of this redesign. DAVE's minimum viable product was also set to be launched later that week.  Within the bullet list I have called out my main struggles during this project.
  • The Tableau dashboard was unresponsive. The tool did not allow for a resizable layout.
  • I had to rely on inserting an image to look like a button within the dashboard. Also, I could not add any alternative text to the picture. This was concerning to me because of accessibility.
  • The data scientists had a lot of content to review. Like 30 pages worth.
Design Impact
I was successful in completing the redesign of the tool. Now, DAVE has 200+ active users building Data visualizations.  Due to the technical limitations of the Tableau platform I had to use images of buttons and icons as links.
Final thoughts
I understand the benefit of DAVE to the MD&I organization, but looking back at the project, I would not recommend the tool being built in Tableau. DAVE is a great product, but Tableau dashboards are not intented to be web sites. The dashboard content is unresponsive and cannot be adjust to fit different device viewports.
